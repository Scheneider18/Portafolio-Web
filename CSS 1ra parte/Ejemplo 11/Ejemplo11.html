Ejemplo de tipos de medios de CSS

Este ejemplo usa tipos de medios de CSS para variar la forma en que aparecer la página impresa y cómo aparece en cualquier otro medio. Este texto aparece con una fuente en la pantalla y con una fuente distinta en papel o en vista previa de impresión. Para ver la diferencia en Internet Explorer, vaya al menú Imprimir y seleccione Vista previa de impresión. Firefox, seleccione Vista preliminar del menú Imprimir.
