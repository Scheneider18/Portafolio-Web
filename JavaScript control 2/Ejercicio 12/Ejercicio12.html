Ejercicios de autoevaluación: 8.12

8.12 Suponga que i = 1, j = 2, k = 3 y m = 2. ¿Qué es lo que imprime cada una de las siguientes instrucciones? ¿Son necesarios los paréntesis en cada caso?

a) document.writeln( i == 1 );

TRUE

b) document.writeln( j == 3 );

FALSE

c) document.writeln( i >= 1 && j < 4 );

TRUE

d) document.writeln( m <= 99 && k < m );

FALSE

e) document.writeln( j >= i || k == m );

TRUE

f) document.writeln( k + m < j || 3 - j >= k );

FALSE

g) document.writeln( !( k > m ) );

TRUE
