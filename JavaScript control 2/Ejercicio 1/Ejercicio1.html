Ejercicios de autoevaluación: 8.1

8.1 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) El caso default es requerido en la instrucción de selección switch.

Falso, el caso default es opcional.

b)La instrucción break es requerida en el último caso de una instrucción de selección switch.

Falso, la instrucción break se utiliza para salir del switch.

c) La expresión ( x > y && a < b ) es verdadera si x > y es verdadera, o si a < b es verdadera.

Falso, ambos operandos deben ser verdaderos para que la expresión sea verdadera.

d) Una expresión que contiene el operador || es verdadera si uno o ambas de sus operandos son verdaderos.

Verdadero
