Ejercicios de autoevaluación: 8.3

8.3 Encuentra el error en cada uno de los siguientes segmentos de código, y explique cómo corregirlo:

a) x = 1;
while ( x <= 10 );
++x;
}

El error se localiza en la linea de while al final se encuentra un punto y coma, para corregirlo se quita el punto y coma y se pone una llave de inicio.

b) switch ( n )
{
case 1:
document.writeln("El número es 1");
case 2:
document.writeln("El número es 2");
break;
default:
document.writeln("El número no es 1 ni 2");
break;
}

El error se encuentra en el case 1 ya que no tiene un break, se soluciona agregando al final del caso 1 un break.

c) El siguiente código debe imprimir los valores 1 a 10:
n = 1;
while ( n < 10)
document.writeln ( n++ )

El error se encuentra en la condición del ciclo se ocupa un operador incorrecto, se soluciona cambiando el < por <=.
