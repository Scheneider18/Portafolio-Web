Ejercicios de autoevaluación: 8.2

8.2 Escriba una instrucción o un conjunto de instrucciones en JavaScript para realizar cada una de las siguientes tareas:

a) Sumar los enteros impares entre 1 y 99. Utilice una instrucción for. Suponga que se han han declarado las variables suma y cuenta.

var suma = 0;
for(var cuenta = 1; cuenta <= 99; cuenta += 2)
suma += cuenta;

b) Calcular el valor de 2.5 elevado a la potencia de 3. Use el método pow.

Math.pow(2.5,3);

c) Imprimir los enteros del 1 al 20 mediante el uso de un ciclo while y la variable x. Suponga que la variable x se ha declarado; pero no se ha inicializado. Imprima solamente cinco enteros por línea.

x = 1;
document.writeln("<p>");
while(x <= 20){
document.write(x + " ");
if (x % == 0){
document.write("</p><p>");
}
++x;
}

d) Repita el ejercicio 8.2, (c), pero ahora use una instrucción for.

x = 1;
document.writeln("<p>");
for (x; x <= 20; ++x){
document.write(x + " ");
if (x % == 0){
document.write("</p><p>");
}
++x;
}
