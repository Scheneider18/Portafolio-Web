3.6 (Formulario de registro Web con encuesta opcional).

Cree un formulario de registro de un sitio Web para obtener el primer nombre, apellido y dirección de correo electrónico de un usuario. Incluya además una pregunta de encuesta opcional que pida el año en el que se encuentra el usuario en la universidad (por ejemplo, primer año). Coloque la pregunta de encuesta opcional en el elementos details que el usuario puede expandir para ver la pregunta.
