3.1 Complete las siguientes oraciones:

a) El tipo input color permite al usuario introducir un color. Al momento de escribir este libro, la mayoría de los navegadores despliegan el tipo input color como un campo de texto en donde el usuario puede introducir un __código hecadecimal__.

b) El atributo __placeholder__ nos permite colocar texto temporal en un campo de texto.

c) Si desea anular la validación, puede agregar el atributo formvalidate al tipo input __submit__

d) El atributo __required__ obliga al usuario a introducir un valor antes de enviar el formulario.

e) Por lo general se muestra el control __spinner__ para el tipo input number e incluye sólo los números válidos.

f) El tipo input __time__ permite al usuario introducir una hora, minuto, segundo y fracción de segundo.

g) El elemento __datalist__ proporciona opciones de entrada para un elemento input text.

h) El elemento __article__ describe contenido que está separada del contenido principal de la página y que posiblemente podría utilizarse o distribuirse en otra parte, como un artículo de noticias, mensaje de foro o entrada de blog.

i) El elemento __footer__ describe el texto que por lo general aparece en la parte inferior del contenido o de un elemento section.

j) El elemento __wbr__ indica el lugar apropiado para dividir una palabra cuando el texto se ajusta en varias líneas.
