3.4 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

Los navegadores que despliegan el tipo input color como campo de texto requieren que el usuario introduzca el nombre de un color.

Falso. Requieren el código hexadecimal en vez del nombre.

b) Cuando un usuario introduce datos en un formulario y luego envía (por lo general, al hacer clic en el boton "Enviar"), el navegador verifica de inmediato que los datos sean correctos.

Verdadero

c) HTML5 puede validar si la dirección de correo electrónico introducida por el usuario existe en realidad.

Falso. Lo unico que puede verificar que siga un formato establecido.

d) Puede agregar required a cualquiera de los tipos input.

Verdadero

e) Puede habilitar autocomplete sólo para ciertos elementos input.

Falso. Se puede habilitar para todos los input o bien para el formulario completo.

f) El elemento time nos permite identificar una fecha, hora o ambas.

Verdadero

g) El elemento caption proporciona una leyenda para la imagen en un elemento figure.

Falso. Para agregar una leyenda a figure se utiliza figcaption.

h) El elemento details muestra una flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, ésta apunta hacia abajo y revela el contenido del elemento summary.

Falso. Los elementos serian a la inversa, summary muestra una flecha y al desplegarla muestra lo almacenado en details.

i) El elemento footer describe el contenido que por lo general aparece en la parte inferior del contenido o del elemento section.

Verdadero

j) El elemento highlight nos permite resaltar el texto.

Falso. El elemento para resaltar el texto es mark.
