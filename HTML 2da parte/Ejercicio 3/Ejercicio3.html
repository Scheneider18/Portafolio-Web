3.3 Complete las siguientes oraciones:

a) El atributo __autofocus__ (que se utiliza en un solo elemento input en un formulario) resalta de manera automática el elemento input y, si se usa de manera apropiada, coloca el cursor en el campo de texto después de que el navegador carga y despliega la página.

b) Los nuevos tipos de input de HTML5 tienen __validación automática__ del lado del cliente.

c) El input __number__ permite al usuario introducir un valor numérico.

d) El input __range__ se valida automáticamente de manera intrínseca cuando lo despliega el navegador como un control deslizante, ya que el usuario no puede mover el control fuera de los límites del valor mínimo o máximo.

e) El atributo __autocomplete__ puede usarse en los tipos input para completar de manera automática la información del usuario con base en datos de entradas anteriores.

f) El elemento __figcaption__ proporciona una leyenda para la imagen en el elemento figure.

g) El elemento summary muestra un flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, ésta apunta hacia abajo y revela el contenido del elemento __details__.

h) El elemento mark nos permite __resaltar el texto__.
