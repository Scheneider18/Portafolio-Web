12.1 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) Cada elemento de HTML5 en una página se representa mediante un árbol DOM.

Falso. Todo elemento se representa mediante un nodo DOM, cada nodo es miembro del árbol DOM.

b) Un nodo de texto no puede tener nodos hijos.

Verdadero

c) El nodo document en un árbol DOM no puede tener nodos hijos.

Falso. El document es el nodo raíz.

d) Es posible cambiar la clase de estilo de un elemento en forma dinámica, estableciendo el atributo style.

Falso. La clase de estilo se modifica estableciendo el atributo class.

e) El método createElement crea un nuevo nodo y lo inserta en el documento.

Falso. El método solo crea el nodo pero no lo inserta en el árbol DOM.

f) El método setInterval llama a una función repetidas veces, con un intervalo de tiempo establecido.

Verdadero

g) El método insertBefore se llama en el objeto documento; recibe un nuevo nodo y uno existente para insertar el nuevo antes.

Falso. insertBefore se llama en el padre.

h) El intervalo más reciente iniciado se detiene al momento de llamar al método clearInterval.

Falso. clearInterval recibe un identificado de intervalo como argumento para determinar qué el intervalo termina.

i) La colección links contiene todos los vínculos en un documento con un atributo id especificado.

Falso. La colección de links contien todos los vínculos en un documento.
