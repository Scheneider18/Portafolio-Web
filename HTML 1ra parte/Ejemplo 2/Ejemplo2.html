2.2 Complete las siguientes oraciones:

a) El elemento __hr__ inserte una regla horizontal.

b) Para mantener un superíndice se utiliza el elemento __sup__ y para marcar un subíndice se utiliza el elemento __sub__.

c) El elemento de encabezado menos importante es __h6__ y elemento de encabezado más importante es __h1__.

d) El elemento __ul__ marca una lista desordenada.

e) El elemento __p__ marca un párrafo.

f) El atributo __type="reset"__ en un elemento input inserta un botón que, cuando se presiona, reestablece el contenido del formularios.

g) El elemento __tr__ marca una fila de una tabla.

h) Los __asteriscos__ se usan por lo general como caracteres de enmascarado en un cuadro de contraseñ.
