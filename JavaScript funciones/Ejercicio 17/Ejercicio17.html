9.17 Implemente las siguientes funciones:

a) La función Celsius devuelve el equivalente en Celsius de una temperatura en Fahrenheir, mediante el cálculo
C = 5.0 / 9.0 * ( F - 32 );
b) La funció Fahrenheir devuelve el equivalente en Fahrenheir de una temperatura en Celsius, mediante el cálculo
F = 9.0 / 5.0 * ( C + 32 );
c) Use estas funciones para escribir una secuencia de comandos que permita al usuario introducir una temperatura en Fahrenheir o Celsius y muestr el equivalente en Celsius o Fahrenheir.
Su documento de HTML debe contener dos botones: uno para iniciar la conversión de Fahrenheir a Celsius y otro para inicar la conversión de Celsius a Fahrenheir.
