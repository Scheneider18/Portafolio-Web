Ejercicios de autoevaluación: 9.25

9.25

Escriba una secuencia de comandos que juegue a "adivinar el número" de la siguiente manera: su programa debe elegir un número a adivinar, para lo cual se selecciona un entero aleatorio en el rango de 1 a 1000. La secuencia de comandos muestra el indicador Adivine un número entre 1 y 1000 enseguida de un campo de texto. El jugador escribe su primer intento en el campo de texto y hace clic en un botón para enviarlo a la secuencia de comandos. Si el intento del jugador es incorrecto, su programa deberá mostrar los mensaje "Demasiado alto. Intente de nuevo" o "Demasiado abajo. Intente de nuevo", para ayudarle a "acercarse" a la respuesta correcta, y deberáborrar el campo de texto para que el usuario pueda introducir el siguiente intento. Cuando el usuario introduzca la respuesta correcta, muestre el mensaje "Felicidades. ¡Adivino el número!" y borre el campo de texto para que el usuario pueda jugar otra vez.
