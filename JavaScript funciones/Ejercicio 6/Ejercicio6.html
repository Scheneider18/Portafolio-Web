9.6

Escriba un programa de JavaScript completo para pedir al usuario el radio de una esfera y luego llame a la función volumenEsfera para calcular y mostrar el volumen de la esfera. Use la instrucción
volumen = ( 4.0 / 3.0 ) * Math.PI * Math.pow( radio, 3 );
para calcular el volumen. El usuario debe introducir el radio en el elemento input de HTML del tipo "number" en un elemento form. Proporcione al elemento input el valor de id "campoEntrada". Puede usar este id con el método getElementById del objeto document para obtener el documento a usar en la secuencia de comandos. Para acceder a la cadena en campoEntrada, use su propiedad value como en campoEntrada.value y luego convierta la cadena en número, usando parseFloat. Use un elemento input de tipo "button" en el elemento form para que el usuario pueda iniciar el cálculo.
