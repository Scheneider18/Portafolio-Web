9.5 Localice el error en cada uno de los siguientes segmentos de programa y explique cómo corregirlo:

a) method g()
{
document.writeln("Dentro del método g");
}

Method no es una palabra clave para iniciar una función, cambiar method por function.

b) // Esta función debe devolver la suma de sus argumentos
function suma (x,y)
{
var resultado;
resultado = x + y;
}

La función no retorna algun valor como esta expecifcado, agregar return resultado; para generar la accion de devolver el valor.

c) function f(a);
{
document.writeln(a);
}

El punto y coma después del paréntesis derecho, se elimina el punto y coma.
