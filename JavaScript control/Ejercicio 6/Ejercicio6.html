Ejercicios de autoevaluación: 7.6

7.6 Determine el valor de cada variable después de realizar el cálculo. Suponga que, cuando se empieza a ejecutar cada instrucción, todas las variables tiene el valor entero 5.

a) producto *= x++;

producto = 25.

x = 6.

b) cociente /= ++x;

cociente = 5/6.

x = 6;
