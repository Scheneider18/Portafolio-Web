7.5 Combine las instrucciones que escribió en el ejercicio 7.4 para formar una secuencia de comandos que calcule e imprima la suma de los enteros del 1 al 10. Use la instrucción while para iterar a través de las instrucciones de cálculo e incremento. El ciclo debe terminar cuando el valor de x se vuelve 11.

<,script type="text/javascript">

var x, suma;
suma = 0;
x = 1;
while(x <= 10){
suma += x;
x = x + 1;
}
document.writeln("<p>La suma de enteros es: "+ suma + "</p>");

</script>
