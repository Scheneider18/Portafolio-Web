10.7 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) Para hcer referencia a una ubicación o elemento específico en un arreglo, específicamos el nombre del arreglo y el valor del elemento.

Falso, para acceder a un elemento específico se especifica el nombre y la posición o índice del elemento.

b) La declaración de una variable reserva espacio para un arreglo.

Verdadero.

c) Para indicar que deben reservarse 100 ubicaciones para el arreglo entero p, el programador debe escribir la siguiente declaración:
p[100];

Falso, para indicar el tamaño de un arreglo debe ser de la siguiente manera var p = new Array(100);

d) Un programa de JavaScript que inicializa los elementos de un arreglo de 15 elementos en cero debe contener cuando menos una instrucción for.

Falso, en la declaración de la variable o arreglo se puede especificar el valor de cada elemento.

e) Un programa de JavaScript que calcula el total de elementos de un arreglo bidimensional debe contener instrucciones for anidadas.

Verdadero.
