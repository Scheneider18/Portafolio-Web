13.2 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) El evento load se dispara cada vez que un elemento empieza a cargarse.

Falso. Se dispara cuando termina de cargarse.

b) El evento click se dispara cuando el usuario hace clic con el raton en un elemento.

Verdadero.

c) El evento focus se dispara cuando un elemento pierde el enfoque.

Falso. Se dispara cuando se gana el enfoque.

d) Al usar el efecto de sustitución con imágenes, es una buena práctica crear objetos Image que realicen cargas previas de las imágenes deseadas.

Verdadero.

e) Al regresar true en un manejador de eventos en un elemento a (ancla) se evita que el navegador siga el vínculo cuando termine el manejador de eventos.

Falso. Es al devolverse false.
