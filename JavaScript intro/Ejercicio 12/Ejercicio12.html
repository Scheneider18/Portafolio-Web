6.12 ¿Cuál de las siguientes instrucciones de JavaScript contiene variables cuyos valores se modifican?

a) p = i + j + k + 7;

La variable p se modifica su valor dependiendo de i, j y k.

b) window.alert("variables cuyos valores se destruyen");

No hay modificaciones.

c) window.alert("a = 5");

No hay modificaciones.

d) cadenaVal = window.prompt("Escriba una cadena");

La variable cadenaVal recibe el valor dado por el usuario.
