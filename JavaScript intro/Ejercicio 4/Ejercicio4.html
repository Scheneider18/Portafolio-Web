6.4 Identifique y corrija los errores en cada una de las siguiente instrucciones:

a) if (c < 7);

window.alert("c es menor que 7");

Error: punto y coma después de la comparación. Corrección: Eliminar punto y coma.

if (c < 7)

window.alert("c es menor que 7");

b) if (c => 7)

window.alert("c es igual o mayor que 7");

Error: El signo de comparación esta al reves. Correción: invertir los signos.

if (c >= 7)

window.alert("c es igual o mayor que 7");
