6.14 Indique el orden de evaluación de los operadores en cada una de las siguientes instrucciones de JavaScript, y muestre el valor de x después de ejecutar cada instrucción.

a) x = 7 + 3 * 6 / 2 - 1;

Orden: *, /, +, -. El valor de x es 15.

b) x = 2 % 2 + 2 * 2 - 2 / 2;

Orden: %, *, /, +, -. El valor de x es 3.

c) x = ( 3 * 9 * ( 3 + ( 9 * 3 / ( 3 ))));

Orden: (), *, /, +. El valor de x es 324.
