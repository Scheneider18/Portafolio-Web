6.11 ¿Qué aparece en el diálogo de alerta cuando se ejecuta cada una de las siguientes instrucciones de JavaScript? Asuma que x = 2 y y = 3.

a) window.alert("x = " + x );

Respuesta: x = 2

b) window.alert("El valor de x + x es " + ( x + x ) );

Respuesta: El valor de x + x es 4

c) window.alert("x = ");

Respuesta: x =

d) window.alert( (x + y ) + "=" + (y + x) );

Respuesta: 5 = 5
