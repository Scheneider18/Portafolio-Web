Ejercicios de autoevaluación: 6.13

6.13 Dada y = ax3 + 7, ¿cuáles de las siguientes instrucciones de JavaScript son correctas para esta acuación?

a) y = a * x * x * x + 7;

Correcta.

b) y = a * x * x * (x + 7);

Incorrecta.

c) y = (a * x) * x * (x + 7);

Incorrecta.

d) y = (a * x) * x * x + 7;

Correcta.

e) y = a * (x * x * x) + 7;

Correcta.

f) y = a * x * (x * x + 7);

Incorrecta.
