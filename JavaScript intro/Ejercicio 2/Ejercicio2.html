6.2 Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

a) Los comentarios hacen que la computadora imprima texto después de los caracteres // en pantalla al momento de ejecutar la secuencia de comandos.

Falso. La computadora no imprime nada que no este dentro de un write o writeln.

b) JavaScript considera que las variables numero y NuMeRo son idénticas.

Falso. Es susceptible a las mayúsculas.

c) El operador residuo (%) puede usarse sólo con operandos numéricos.

Verdadero

d) Los operadores aritméticos *, /, %, + y - tienen todos el mismo nivel de precedencia.

Falso. Los operadores *, / y % estan en el mismo nivel y + y - estan un nivel abajo.

e) El método parseInt convierte un entero en una cadena.

Falso. El método funciona al revés.
